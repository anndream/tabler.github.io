CSS3 Form Components

We used CSS3 for most of our components to avoid using unnecessary JavaScript libraries. All this to make it fast, reliable and pleasant to use by everyone!

Build with Bootstrap

Tabler used Bootstrap, the most popular HTML, CSS, and JS framework. Thanks to this developers without experience can modify code without any problems.

Simply, beautiful, open source icons

Tabler is build with the newest version of Feather icons made by Cole Bemis. Each icon is designed with an emphasis on simplicity, consistency and readability.
